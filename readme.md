# Neovim Configuration

Neovim configuration with plugins and settings.

## Requirements

nvim version >= 0.12.0

## Install Dependencies

```shell 
npm i -g vscode-langservers-extracted typescript-language-server typescript @prisma/language-server
```

## Structure

- `lua/plugins/` - plugins
- `lua/config/` - settings (options, keybinds, lsp, diagnostic)
- `lua/custom_plugins/` - custom plugins
- `lua/themes/` - themes
- `lsp/` - LSP servers
